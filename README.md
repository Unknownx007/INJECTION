## ⚙️ How It Works (The Logic Matrix)

1. **DOM Scrape Pass:** The program reads your target URL and isolates all text fields (mapping complex IDs like `ctl00$txtUserName`).
2. **Baseline Handshake:** It executes a neutral connection pass to capture the original baseline size of the application layout.
3. **Parallel Probing:** The prober fires specific raw syntax break symbols (`'` and `"`) straight into each field independently while auto-filling mandatory background fields.
4. **Variance Detection:** If an input lane accepts raw special characters, the server's internal database processing behavior changes. Even if it blocks visible error printouts, the final page size drifts from the baseline. If a shift of **even 1 byte** occurs, the engine flags the parameter as **`SYMBOLS ALLOWED (POSSIBLY VULNERABLE)`**.
